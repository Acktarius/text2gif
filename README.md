# text2gif
convert a text in an animated gif from right to left like a *marquee* html style

## Requirement:  
image Magick :  
`sudo apt update`  
`sudo apt install imagemagick`  

## install:  
`git clone https://github.com/Acktarius/text2gif.git`  
`cd text2gif`  
`chmod 755 text2gif.sh`  


## use: 
`cd text2gif`   
`./text2gif.sh`
